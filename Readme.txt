Topic : EEG Singnal Data VIsualization using Streamlit Framework


WEB APPLICATION :
	
	* Data Visualization based EEG Signal Dataset.
	*Channel Based data visualization.
	* Frequency Selection based data visualization.

RUNNING METHODOLOGY :

	*Copy of our Project Directory .
	*Open Anaconda Prompt.
	*Type  'cd <paste your project directory>" to combine our path into anaconda.
	*Type "streamlit run app.py" to start the development server.
	*copy the Web url and paste any browser to show the result.
